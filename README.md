# Universal Video Downloader ユーザーマニュアル＆仕様書

## [English](README_ENG.md)

このアプリケーションは、yt-dlp・aria2c・ffmpegを利用し、最高速度と利便性を追求して設計された高機能な動画ダウンローダーです。

---

## 🌟 主な機能と特徴

### 1. 超高速ダウンロード（最適化仕様）
- **aria2c連携**: 1つのファイルに対して最大32コネクション（`-x 32 -s 32 -j 32`）を張り、爆速でダウンロードします。
- **並列フラグメント**: yt-dlpのフラグメント並列ダウンロード（32並列）に対応し、HLSやDASH配信の動画も高速化。
- **ffmpegマルチスレッド処理**: お使いのPCのCPUコア数を自動で検知し、動画と音声の結合・エンコード処理をマルチスレッド（`-threads N`）でフル稼働させます。
- **SSD最適化**: aria2cの `--file-allocation=none` オプションにより、ダウンロード開始時の不要なディスク割り当て待ちをスキップしSSDの寿命と速度に貢献します。

### 2. 強力なサイト対応と回避能力
- **対応サイト**: YouTube, TikTok, ニコニコ動画, X (Twitter), SoundCloud, Bilibili, Twitch, Instagram 等（yt-dlpが対応する数千サイト）
- **Shorts・ライブアーカイブ完全対応**
- **YouTube 403 Forbidden回避**: 最新の制限を回避するため、内部でスマートフォン版（iOS）クライアントを偽装してダウンロードを試行します。
- **Cookie全自動切り替え**: 
  1. まずCookieなしで安全にダウンロードを試みます。
  2. 失敗した場合、お使いのPCのブラウザ（`Chrome` → `Edge` → `Firefox` → `Brave`）の順で自動的にCookieを読み込み、年齢制限・会員限定動画のダウンロードを再試行します。

### 3. スマートなUIと全自動機能（待機モード）
- **完全日本語・ダークモードUI**: 目に優しくスタイリッシュなCustomTkinter製GUI。
- **クリップボード自動監視**: ブラウザで動画のURLを「コピー」するだけで、アプリの入力欄に自動でペーストされます。
- **待機モード（全自動ダウンロード）**: 「待機モード」をONにしておくと、URLをコピーした瞬間に**クリック不要で全自動でダウンロード**が開始されます。連続で保存したい時に最強の機能です。

### 4. 拡張・保守機能
- **ダウンロード履歴**: 過去のダウンロード履歴をSQLiteデータベースで記録し、「履歴」タブからいつでも確認可能。
- **自動アップデート**: 「更新確認」ボタンから、ダウンロードのコアエンジンである `yt-dlp` を最新版にワンクリックでアップデートできます。（動画サイトの仕様変更によるダウンロード失敗を即座に直せます）
- **ホットパッチ機能**: exe化した状態でも、`patches` フォルダにPythonスクリプトを入れるだけで、有志が作成したパッチや機能追加を後から適用できます。

## 🛠️ 事前準備 (必須環境)

本ソフトは内部で以下の強力な外部ツールを使用しています。あらかじめインストールし、システムの**環境変数（PATH）**に通しておく必要があります。

1. **ffmpeg**: 動画と音声の結合・フォーマット変換に必須です。
2. **aria2c**: 超高速32並列ダウンロードを行うためのエンジンです。

## 📖 使い方

### 基本的なダウンロード
1. ソフト（`UniversalVideoDownloader.exe` 又は `python main.py`）を起動します。
2. ダウンロードしたい動画のURLをコピーします（自動で入力欄に入ります）。
3. 入力欄の下にあるドロップダウンからお好みの設定を選びます：
   - **画質**: 最高画質 / 1080p / 720p / 360p
   - **音質**: 最高音質 / 320kbps / 256kbps / 128kbps （※音声抽出時のみ有効）
   - **拡張子**: mp4 / mkv （動画用） または mp3 / flac / wav / mka （音声用）
4. **「ダウンロード開始」** をクリックします。

### 待機モードの使い方
「ダウンロード開始」ボタンの横にある「待機モード: OFF」をクリックしてON（赤色）にします。
この状態で、ブラウザやX(Twitter)のアプリで「リンクをコピー」するだけで、バックグラウンドで次々と指定した画質・フォーマットでダウンロードが行われます。
> ※ダウンロード中に新しいURLをコピーした場合は、安全のため自動開始がスキップされます。

### エラーが出た場合
ログ画面にエラーが出た場合は以下をお試しください。
- **更新確認ボタンを押す**: yt-dlpを最新にすることで直るケースが9割です。
- **ブラウザを閉じる**: 会員限定動画などでCookie読み込みに失敗している場合、対象のブラウザ（Chromeなど）を完全に終了させてからダウンロードを行うと、Cookieのロックが解除されて成功します。
